PowerBI API Server

Sample API Server for report embedding services.

This json api could be consumed by various types of clients

Example requests:

  • All reports
    /api/reports

  • Find single report
    /api/reports/381b4400-0836-47f3-81e3-0979f830b7d9
    (Note: This is just a sample Report ID that has been uploaded previously into Azure.)
    (Refer to https://github.com/Microsoft/PowerBI-Cli for information on how to create Workspaces and upload Reports using PowerBI CLI.)
    (Refer to https://github.com/Microsoft/PowerBI-Node for information on how to use PowerBI API - Server)
    (Refer to https://github.com/Microsoft/powerbi-javascript for information on how to use PowerBI API - Client - Pure Javascript)
    (Refer to https://github.com/Microsoft/PowerBI-Angular for information on how to use PowerBI API - Client - Angular)
